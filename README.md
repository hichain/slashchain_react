## 対応ゲーム

- slashchain (制作中)
- HICHAIN (予定)

## Usage

### Installation

```
$ npm install
```

### Running Game Client

```
$ npm start
```

### Running Game Server

```
$ npm run serve:websocket
```

### Running Tests

```
$ npm test
```

### Running Linter

```
$ npm lint
```
