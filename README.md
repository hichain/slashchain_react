## 対応ゲーム

- slashchain (制作中)
- HICHAIN (予定)

## Usage

### Installation

```
$ yarn
```

### Running Game Client

```
$ yarn run start:client
```

### Running Game Server (local)

```
$ NODE_ENV=development yarn run serve:master
```

### Running Tests

```
$ yarn test
```

### Running Linter

```
$ yarn lint
```
