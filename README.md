## Usage

### Installation

```
$ yarn
```

### Running Game Client

```
$ yarn start:client
```

### Running Game Master Server (local)

```
$ yarn start:master
```

### Running Tests

```
$ yarn test
```

### Running Linter

```
$ yarn lint
```
